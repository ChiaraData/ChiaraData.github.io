Analyse et prévision du risque de défaut d'entreprises locales Aix/Marseille

Contexte et Objectifs

Ce projet vise à répondre à un défi majeur du secteur financier : l'anticipation du risque de défaillance des entreprises. L'objectif initial était d'exploiter les bilans financiers publics pour construire un score de risque prédictif. Face aux contraintes d'accessibilité des données comptables (APIs payantes), j'ai pivoté vers une approche par signaux "Proxy" exploitant les caractéristiques structurelles et les annonces légales pour évaluer la santé des entreprises locales.

Réalisé par : Chiara Masi
Période : Juillet 2025 - Janvier 2026.

Compétences et Outils Utilisés

Python (Pandas, Numpy) Scikit-learn (Random Forest) SQL MySQL / MariaDB Visualisation (Matplotlib, Seaborn) Ingénierie de données & ETL

Sources de données : API SIRENE (INSEE) pour le profilage, API BODACC (annonces légales) pour l'identification des défauts, et indicateurs macro-économiques régionaux (PACA) pour le contexte conjoncturel.

Approche Méthodologique et Modèles

L'architecture du projet repose sur une séparation entre la collecte (ETL) et l'analyse prédictive :

  • Ingestion de données : Développement de scripts Python pour l'alimentation automatisée d'une base MySQL.
  • Feature Engineering : Création de variables clés telles que l'ancienneté de l'entreprise et l'encodage des secteurs d'activité (NAF).
  • Traitement du déséquilibre : Utilisation d'un modèle Random Forest avec pondération des classes (class_weight='balanced') pour traiter un dataset comportant moins de 1% de défauts.

Résultats Clés

Le modèle a permis de générer un Score de Risque sur 100 pour plus de 20 000 entreprises de la zone Aix-Marseille. Les analyses démontrent que l'âge de l'entreprise et son secteur géographique sont des prédicteurs majeurs en l'absence de données comptables.

Note d'expérience : Ce projet illustre ma capacité à surmonter les limitations techniques et budgétaires en adaptant la stratégie de modélisation pour extraire de la valeur à partir de données alternatives.
